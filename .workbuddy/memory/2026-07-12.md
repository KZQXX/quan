### 关键技术决策
- JWT Token 有效期由 `settings.jwt_expire_minutes` 配置（默认 60 分钟）
- JWT 算法由 `settings.jwt_algorithm` 配置（默认 HS256）
- JWT 密钥由 `settings.jwt_secret` 配置（dev 默认值，生产必须覆盖）
- bcrypt 使用标准 gensalt()，无额外配置
- 所有模型主键使用 UUID v4（UUIDMixin），时间戳使用 TimestampMixin
- 所有 API 基于 FastAPI 类型化依赖注入（Annotated + Depends）
